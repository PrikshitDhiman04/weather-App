# Weather App

This is a basic **Weather Application** built using HTML, CSS, and JavaScript.  
The app allows users to search for a city and view current weather information.

This project was created to practice working with APIs and handling user input in JavaScript.

---

## Features

- Search weather by city name
- Display temperature and weather condition
- Simple and clean interface
- Real-time data using a weather API

---

## Technologies Used

- HTML
- CSS
- JavaScript
- Weather API
